C1 Schrift 写作框架

C1 Schrift

写作总体框架

Einleitung (50-60 Wörter)

  • Aufhänger

    • Heutzutage wird kontrovers über … diskutiert.
    • In den vergangenen Tagen wurde häufig über … diskutiert
    • Laut einer aktuellen Studie …
    • Sollen … oder nicht?

  • Fragestellung

    • In diesem Zusammenhang taucht immer wieder die Frage auf, ob …
    • In dieser Situation stellt sich die Frage, ob es nicht besser wäre, wenn …
    • Daraus ergibt sich die Frage …

  • Weiteres Vorgehen nennen

    • Im Anschluss daran …
    • Abschließend …
    • Im Folgenden soll … erörtert werden

Hauptteil (240 Wörter)

TAB These Argument Beispiel
Pro 1 These Argument Dafür lassen sich viele Beispiele nennen, wie …
Pro 2 These Argument Befürworter dieser Ansicht bedienen sich häufig der folgende Argument: …
Contra 1 These Argument Gegen diese Argumentation ist allerdings einzuwenden, dass …
Contra 2 These Argument Der wichtigste Einwand bezieht sich auf …

Im Schluss

  • Fazit

    • Alles in Allem zeigt sich, dass …
    • Betrachtet man die Argumente, muss man zum Schluss kommen, dass …
    • Ich sehe auf beiden Seiten wichtige Argumente. Deshalb komme ich zu dem Ergebnis, dass …
    • Um abschließend zur Ausgangsfrage zurückzukommen: …

  • Eigene Meinung

    • Ich beurteile dieses Problem folgendermaßen.
    • Ich vertrete den Standpunkt, dass …
    • Ich bin davon überzeugt, dass …
    • Ich bin mir unsicher, ob …
    • Ich bezweifle, dass …
